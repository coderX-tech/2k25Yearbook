## üöÄ QUICK START

### Backend (Django)
```bash
cd yearbookBackend
pip install -r requirements.txt
python manage.py makemigrations
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```

### Frontend (React)
```bash

npm install
npm run dev
```
